a5.phlocalities Installation
----------------------------

To install a5.phlocalities using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``a5.phlocalities`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        a5.phlocalities

* Re-run buildout, e.g. with:

    $ ./bin/buildout
